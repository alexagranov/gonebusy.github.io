Types of Pricing Models

Pricing Models are associated with Services in order to determine how bookings for the Service will be billed. Additionally, the rate defined by the Pricing Model is shown to customers when viewing advertised Services.

There are currently 4 different types of Pricing Models supported, each of which are described below.

Activity

– represents a Service that is offered free of charge. Examples of this type of Service would be Meetings, Book Club, Girls Night Out, etc. By default, when Services are created without specifying an existing Pricing Model, an Activity Pricing Model is created and associated with the Service.

By The Hour

– used to specify the hourly rate to charge for Services. Services may still be booked for increments of less or more than an hour, if so desired.

By The Minute

– used to specify the minute rate to charge for Services. Use this Pricing Model for Services that require fine-grained billing such as Customer Support, Incident Response, etc.

Fixed Price

– used to specify a fixed rate to charge for Services, regardless of the length of the booking. This pricing model is useful for Services that represent projects or jobs that require a flat fee.

If you need to price a Service in a manner that is not accommodated by any of the currently available PricingModels, please let us know and weâ€™ll be happy to assist you in creating a new PricingModel if necessary.

Activity

Sample POST request to create an Activity PricingModel:

Request:

{% highlight ruby %} http://beta.gonebusy.com/api/v1/pricing_models/new.json {% endhighlight %}

with POST body:

{% highlight ruby %} "type": "Activity", "name": "Weekly Update Meeting", {% endhighlight %}

Response:

{% highlight ruby %} "Status": "201 Created" { "pricing_model": { "id": 125, "owner_id": 54, "name": "Weekly Update Meeting", "notes": "", "pricing_model_type": "Activity" } } {% endhighlight %}

By The Hour

Sample POST request to create a ByTheHour PricingModel:

Request:

{% highlight ruby %} http://beta.gonebusy.com/api/v1/pricing_models/new.json {% endhighlight %}

with POST body:

{% highlight ruby %} "type": "ByTheHour", "price": 50.0, "name": "Tutoring Hourly Rate", "notes": "Only portions of an hour are billed." {% endhighlight %}

Response:

{% highlight ruby %} "Status": "201 Created" { "pricing_model": { "id": 126, "owner_id": 54, "name": "Tutoring Hourly Rate", "price": 50.0, "currency": "USD", "notes": "Only portions of an hour are billed.", "pricing_model_type": "ByTheHour" } } {% endhighlight %}
