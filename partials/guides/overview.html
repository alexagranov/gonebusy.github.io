Overview

The world of GoneBusy revolves around:

  • Users – a Person or Business registered to use, or managed by, the API

  • A User's Resources – such as Meeting Rooms, Equipment, individual Staff members, etc.

  • The Services offered by Resources – "Weekly Status Meeting", "Guitar Lessons", "Rent My Parking Space", etc.

  • The Schedules of Services.

  • The Booking of a Service at a particular time

  • Search for available Services.

The GoneBusy API allows you to programmatically create and manage all of the above items as well as more advanced features such as Categories and Pricing Models.

All Users automatically gain access to the GoneBusy API. The API allows for the creation and management of the Userâ€™s Resources, Services, Schedules and Bookings.

The API Key for access can be found under the "API Key" menu option on the User's dashboard.

Account Manager

The API allows creation of other User accounts for which the User becomes the defacto Account Manager. The Account Manager User can then create and manage Resources, Services, and Schedules of the Users for which they are the Account Manager. Account Managers have only limited access to manage the Bookings of their managed Users.
