Overview

The world of GoneBusy revolves around:

  • Users and their Resources

  • the Services/Activities the above two offer

  • the Schedules of Services/Activities

  • the Booking of a Service at a particular time

  • customers can Search for any available Services/Activities

The GoneBusy API allows you to programmatically create and manage all of the above items as well as more advanced features such as Categories and Pricing Models.

You can jump right in and explore the GoneBusy API using our Swagger UI API Sandbox.

Basic Users

When you first sign up, you are a Basic User. A Basic User cannot create any Resources, the only Resource they have is themselves. A Basic User can create Services/Activities but they must be offered at no charge. Basic User accounts are free for the first year, and $1 per year after the first year.

Pro Users

Any Basic account can be upgraded to a Pro account. Pro Users are able to create additional Resources under their account. These Resources can have their own Schedules as well as Services/Activities offered. Pro Users are also able to offer Services/Activities at a price to the customer. The price of a Service/Activity can be fixed for the duration of the appointment, hourly, or various other options. Pro User accounts are still free for the first year and $1 per year after the first year. GoneBusy does charge a nominal 2.5% + 50¢ fee of a successfully completed Booking (see description of Bookings).

Account Manager

The API allows creation of other accounts for which the Pro User becomes the defacto Account Manager. The Pro User, via API, can then create and manage Resources, Services/Activities, and Schedules of Users for which they are the Account Manager. Account Managers have only limited access to manage the Bookings of their managed Users.

Any Users created via API are completely viewable and have their Services and Schedules accessible by customers on the GoneBusy website.

The Affiliate Program

By prior written arrangement, a Pro User can become an Affiliate of GoneBusy. If any Bookings are initiated on the GoneBusy website for Users created by Affiliates, the Affiliate receives a commission on every successfully completed Booking (see description of Bookings).
