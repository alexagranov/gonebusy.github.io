Categories are a way of organizing or tagging available Services.

Get Categories

This will look up all Categories in the system. If retrieving Categories for a particular User, you must be authorized to manage this User.

{% highlight ruby %} GET /categories {% endhighlight %}
Parameter Type Description Required?
user_id Integer Provide a User ID to see only the Categories of Services offered by a particular User
page Integer Result page of interest, default: 1
per_page Integer Number of results per page, default: 10

Get Category by ID

Look up details for a Category by ID. You can access details for any Category.

{% highlight ruby %} GET /categories/{id} {% endhighlight %}
Parameter Type Description Required?
id Integer ID of category to retrieve

Create a Category

Create a new Category. Please note that once created, it can not be modified. A duplicate named Category under the same Parent Category ID is not permitted.

{% highlight ruby %} POST /categories/new {% endhighlight %}
Parameter Type Description Required?
name String Name of new Category
description String Description for Category
short_name String Abbreviated name for Category
long_name String Full name of Category
parent_category_id Integer ID of parent Category
