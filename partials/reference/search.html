Search

GoneBusy offers a fully featured Search interface for retrieving Services/Activities that meet search criteria such as Category name, geographic distance, rating, price, etc.

The following endpoint can be used to perform all Searches:

Get Search Results

Get Search Results

All Service/Activites that match given search criteria will be returned.

{% highlight ruby %} GET /search/{query} {% endhighlight %}
Parameter Type Description Required?
query String Search query string
page Integer Result page of interest, default: 1
per_page Integer Number of results per page, default: 10
