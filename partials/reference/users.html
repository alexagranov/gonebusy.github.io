Users

Users are the people that participate on GoneBusy. If you haven't done so already, take a look at our explanation of how Users interact with GoneBusy on the "How GoneBusy Works" page.

The API allows a User to manage themselves or an Account Manager to operate on managed users in order to offer services, schedule availability, and various other operations.

Get Users

This will look up all Users that your account has access to - by default this includes all Users under your own User account as well as any Users of Users for which you are the Account Manager.

{% highlight ruby %} GET /users {% endhighlight %}
Parameter Type Description Required?
page Integer Result page of interest, default: 1
per_page Integer Number of results per page, default: 10

Get User by ID

Look up details for a User by ID. You can access details for any User under your own User account as well as any Users of Users for which you are the Account Manager.

{% highlight ruby %} GET /users/{id} {% endhighlight %}
Parameter Type Description Required?
id Integer ID of User to retrieve

Create a User

Create a new User. By default, your own User account will automatically become the Account Manager for any newly created Users.

{% highlight ruby %} POST /users/new {% endhighlight %}
Parameter Type Description Required?
email String Email address
first_name String First name
last_name String Last name
business_name String Name of business/organization
external_url String Website URL
permalink String Vanity URL at GoneBusy - ex: www.gonebusy.com/<permalink> - must be unique
timezone String Timezone acronym
